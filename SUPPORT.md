# ScannrExpense Support

For help with ScannrExpense, contact: support@scannrfast.se

Please include:

- The device model and iOS version.
- The ScannrExpense app version and build number.
- A short description of the problem.
- Screenshots if they help explain the issue, but avoid sending sensitive receipt data unless needed.

## Common checks

- Confirm that iCloud is enabled if you expect data to sync between devices.
- Confirm that your subscription is active in Apple ID purchase settings if Pro features are unavailable.
- Restart the app and try the action again before contacting support.
